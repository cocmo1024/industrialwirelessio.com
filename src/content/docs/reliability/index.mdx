## Reliability review flow

<Steps>
	1. List the most likely environmental, power, and communications failure modes.
	2. Decide which failures must be tolerated locally and which require intervention.
	3. Design maintenance, recovery, and replacement paths before deployment.
	4. Re-check whether the architecture is still reasonable after those realities are included.
</Steps>
