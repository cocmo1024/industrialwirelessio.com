## Protocol review flow

<Steps>
	1. Confirm what happens at the field device layer and what happens upstream.
	2. Decide where translation, buffering, and data shaping should occur.
	3. Choose the simplest protocol mix that still supports the monitoring objective.
	4. Avoid treating every layer of the architecture as if it needs the same protocol.
</Steps>
