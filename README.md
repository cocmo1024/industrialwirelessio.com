# Industrial Wireless I/O

Static Astro site for [industrialwirelessio.com](https://industrialwirelessio.com/).

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Foundation

- Homepage
- About page
- Contact page
- Privacy policy
- Disclaimer
- Sitemap, robots, RSS, and base SEO metadata
